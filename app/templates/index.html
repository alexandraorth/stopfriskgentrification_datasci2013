The purpose of our project is to explore the relationship between gentrification and stop and frisk in New York City. Our working hypothesis was that newly gentrifying areas have increased police presence and that this increased police presence results in a greater number of stop and frisk incidents.

Constructing the Gentrification Index

To measure gentrification, we constructed an index. Our methodology for constructing the index was inspired by the method used by Nancy Hudspeth, et al (CITE). We then calculated the percent change for each community district between 2000 and 2010. Next, the percent changes for each variable for each district were compared to the mean NYC percent change for each variable for each district. However, we realized that gentrification might vary between boroughs, so we decided to recalculate the gentrification scores for each district by comparing the variablesâ€™ percentage changes to the average percentage changes for the borough.

Summarize, Communicate, Build

The total number of stops across all community districts in New York City was displayed using a choropleth map of the boroughs. This map was built using D3.js and jQuery. The geoJSON boundary data was taken from from NYC Open Data and then converted into topojson and merged with district numbers. The map was colored according to a scale developed around the stop and frisk data from the New York Police Department. The same process was applied for the gentrification map, using the calculated gentrification index for coloring.

The scatterplot was also created using D3.js. We have plotted the normalized total number of stops versus the gentrification index, which shows an increased number of stops as the area becomes more gentrified. The size of the circles represents the arrest to stop ratio for each specific gentrification index.
